An (unfinished) unobtrusive IRC client for Windows.

Download (57 KB 7z, .Net 3.5 required, code included)

Known issues

  • Lesser-known protocol features are not implemented (or untested).
  • No GUI for user info, channel options and such. Type commands for now.
  • RichTextBox is used for log display (this is temporary).
  • No proxies and SSL (some argue it's pointless anyway).
  • Mirc text colors and formatting are not supported.
