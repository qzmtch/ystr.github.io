Nawtpad α

A half-assed code editor.

  • No modal dialogs, all tools at hand.
  • Incremental search, occurrence marking.
  • Overview scrollbar.
  • Brace matching.
  • Syntax highlighting.
  • Autoindentation on paste.
  • Edit one file in multiple windows.
  • Themes, schemes.
  • Comment/uncomment.
  • Bookmarks.

Download (100 KB, .Net 3.5 required, code included)

Note: run ngen install Nawtpad.exe to reduce startup time. There is also an option to Remain in memory for faster opening.

Known bugs and limitations

  • Proportional fonts are not supported.
  • Combining characters are rendered as separate characters.
  • May contain serious issues with some encodings (untested).
  • No right-to-left etc.
